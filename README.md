# 🛒 C++ E-Ticaret Sistemi

![C++](https://img.shields.io/badge/C%2B%2B-Programming-blue)
![Project](https://img.shields.io/badge/Project-E--Commerce-green)
![Status](https://img.shields.io/badge/Status-Completed-success)
![License](https://img.shields.io/badge/License-Educational-lightgrey)

Bu proje **C++ programlama dili kullanılarak geliştirilmiş bir konsol tabanlı e-ticaret sistemi simülasyonudur.**
Kullanıcıların ürünleri görüntüleyebildiği, sepete ekleyebildiği ve müşteri bilgilerini yönetebildiği basit bir alışveriş sistemi içerir.

---

# 🎯 Proje Amacı

Bu proje aşağıdaki yazılım geliştirme becerilerini geliştirmek amacıyla hazırlanmıştır:

* Nesne Yönelimli Programlama (OOP)
* Dosya okuma ve yazma işlemleri
* Konsol tabanlı uygulama geliştirme
* Temel e-ticaret sistemi mantığını modelleme

---

# ⚙️ Kullanılan Teknolojiler

* 💻 **C++**
* 📂 **Dosya İşlemleri (File Handling)**
* 🧱 **Nesne Yönelimli Programlama (OOP)**

# 📂 Proje Yapısı

Projede bulunan bazı önemli dosyalar:

```
main.cpp
Urun.h
Sepet.h
Musteri.h
MusteriOkuma.h
Kategoriler.h
musteri.txt
```

### Dosyaların Görevleri

| Dosya          | Açıklama                         |
| -------------- | -------------------------------- |
| main.cpp       | Programın başlangıç noktası      |
| Urun.h         | Ürün sınıfını tanımlar           |
| Sepet.h        | Sepet işlemlerini yönetir        |
| Musteri.h      | Müşteri sınıfını tanımlar        |
| MusteriOkuma.h | Müşteri verilerini dosyadan okur |
| Kategoriler.h  | Ürün kategorilerini yönetir      |

# ▶️ Programı Çalıştırma

### 1️⃣ Repoyu klonlayın

```
git clone https://github.com/kullaniciadi/projeadi.git
```

### 2️⃣ Proje klasörüne girin

```
cd projeadi
```

### 3️⃣ Programı derleyin

```
g++ main.cpp -o eticaret
```

### 4️⃣ Programı çalıştırın

```
./eticaret
```
